# Computer-Graphic
Программа реализована на языке Java, использовались Scene и SceneBuilder.

Реализованы алгоритмы согласно варианту задания.
Предусмотрены:
1) Выведена система координат.
2) Отображены окна отсекающие одним цветом, исходные отрезк(многоугольники) – другим цветом.
3) Выполнены отсечение соответствующими алгоритмами.
4) Визуализированы видимые части отрезков (многоугольников).

Использование программы:
1) Запускаем jar, выбираем алгоритм по центру в списках
2) Нажимаем красную кнопку File и выбираем один из входных файлов подготовленных заранее
3) Жмём кнопку cat и получаем результат, который будет подсвечен другим цветом.
