# Computer-Graphic
Программа реализована на языке Java, использовались Scene и SceneBuilder.

Реализованы алгоритмы согласно варианту задания.
Предусмотрены:
1) Визуализация объекта
2) Трёхмерные преобразования
3) Построение проекций
4) Вывод матриц преобразования

Использование программы:
1) Запускаем jar, выбираем алгоритм по центру в списках
2) Для рассмотрения объекта с каждой стороны достаточно нажать в любой области экрана и повернуть в желаемом направлению наш объект. При помощи мыши также можно приблизить фигуру. 
3)Для преобразовния необходимо также нажать соответствующую кнопку и внести нужные данные. В результате отобразится преобразованный объект, при условии корректности ввода.
4)Для построения проекций достаточно нажать на соответствующие кнопки
